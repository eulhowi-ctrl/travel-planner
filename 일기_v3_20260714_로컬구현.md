#### 2️⃣ 백엔드 구현
**선택**: backend/app/main.py (FastAPI - 완전 기능)

**구조**:
```
backend/app/
├── main.py              (FastAPI 메인 서버)
├── models.py            (SQLAlchemy ORM - 9개 모델)
├── routers/
│   ├── destinations.py  (여행지 API)
│   ├── users.py         (사용자 API)
│   ├── itineraries.py   (일정 API)
│   ├── budgets.py       (예산 API)
│   ├── transportation.py (교통 API)
│   └── qrcodes.py       (QR코드 API)
├── utils/
│   ├── seed_data.py     (여행지 데이터)
│   └── qr_generator.py
└── database.py          (DB 설정)
```

**API 엔드포인트**:
- `GET /api/v1/destinations` - 여행지 목록 (350개)
- `GET /api/v1/destinations/{id}` - 특정 여행지
- `GET /api/v1/countries` - 나라 목록
- `POST /api/v1/itineraries` - 일정 생성
- `POST /api/v1/budgets` - 예산 추가
- 기타 사용자, 교통, QR코드 API
